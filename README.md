# Instagram Influencer Performance Analysis

This project analyzes Instagram post engagement to understand what drives higher audience interaction and long-term content value. Using cleaned Instagram data, key metrics such as engagement rate, impressions, and content value are analyzed.

An interactive Power BI dashboard visualizes insights related to hashtags, captions, likes, saves, and overall engagement to help influencers and marketers optimize their content strategy.

**Tools Used:** Python, Pandas, Power BI
**Domain:** Data Analytics / Social Media Analytics
